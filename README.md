# inflation-forecasting-dissertation
This repository contains the code, analysis and dissertation report for the undergraduate research project for BSc (Hons) Statistical Data Science in Heriot Watt University

# Dissertation Title
Univariate vs. Multivariate Forecasting Approaches: A Comparative Study of Time Series and Machine Learning Models for Inflation Prediction

# Project Overview
The stability and growth of the economy are largely influenced by the expectations formed
by policymakers, making reliable inflation forecasting a critical component of economic planning.
This research examines the limitations of commonly used forecasting models and compares the performance of alternative approaches. Both univariate and multivariate methods
from time series, including Single Exponential Smoothing, Double Exponential Smoothing,
ARIMA and VAR, and machine learning techniques, including k-Nearest Neighbors, Random
Forest and LSTM, are applied to construct the forecasting of two CPI components, using historical macroeconomic data from the US government. The comparison emphasizes predictive
accuracy across multiple forecast horizons, with particular focus on model feature selection,
stability, and the ability to capture nonlinear patterns and structural changes in the data.

# Models Evaluated

### Time Series Models

- ARIMA
- GARCH
- Exponential Smoothing
- VAR


### Machine Learning Models

- k Nearest Neighbors (kNN)
- SVR
- Random Forest
- Long Short-Term Memory Networks (LSTM)

# Research Objectives
1. To compare univariate and multivariate approaches using reliable, consistent performance metrics.
2. Investigate model performance across different forecasting horizons.
3. Assess the trade-off between predictive accuracy, model complexity, and interpretability.
4. Indentify the effect of the presence of randon fluctuations in the data to prdictive performance.
